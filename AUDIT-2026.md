## 3. Аудит тестового покрытия

### 3.1. Критическая проблема

**Статус**: ❌ Тесты не запускаются

**Команда**: `npm run test -- --run`

**Результат**:
```
 FAIL  tests/cutting/cutting.test.ts
 FAIL  tests/geometry/bbox.test.ts
 FAIL  tests/geometry/curves.test.ts
 FAIL  tests/geometry/math.test.ts

Error: No test suite found in file
```

**Причина**: Конфигурация Vitest с `globals: true` требует глобальных `describe`, `it`, `expect`, но они не распознаются в текущей версии.

**Решение**: Добавить явные импорты в начале каждого тестового файла:
```typescript
import { describe, it, expect, beforeEach, afterEach } from 'vitest';
```

Или обновить конфигурацию vitest.config.ts:
```typescript
export default defineConfig({
  test: {
    globals: true,
    // Добавить явно
    setupFiles: ['./tests/setup.ts'],
  },
});
```
