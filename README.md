# DXF Viewer — Просмотр чертежей с анализом лазерной резки

Веб-приложение для просмотра DXF-файлов с функциями анализа лазерной резки и раскладки деталей на листы металла.

## Возможности

### Просмотр DXF

- **Парсинг ASCII и Binary DXF** — поддержка версий от R12 до R2021
- **24 типа сущностей** — LINE, ARC, CIRCLE, ELLIPSE, SPLINE, LWPOLYLINE, POLYLINE, POINT, SOLID, TRACE, HATCH, TEXT, MTEXT, DIMENSION, LEADER, MLEADER, INSERT, ATTDEF, ATTRIB, 3DFACE, IMAGE, UNDERLAY, VIEWPORT, XLINE/RAY
- **Рекурсивное разворачивание INSERT** — блоки с вложенностью, массивы, трансформации
- **Парсинг в Web Worker** — загрузка не блокирует интерфейс

### Навигация

- **Zoom колёсиком** — приближение/удаление к точке курсора
- **Pan мышкой** — перетаскивание зажатой кнопкой
- **Fit (F)** — вписать чертёж в экран
- **Сетка (G)** — включение/выключение координатной сетки

### Мультифайловая работа

- **Загрузка нескольких DXF** — через диалог, drag-n-drop или Ctrl+O
- **Список файлов** — боковая панель с переключением между файлами
- **Чекбоксы** — включение/выключение файлов для расчёта резки
- **Удаление файлов** — из списка по отдельности
- **Прогресс-бар** — с именем загружаемого файла

### Анализ лазерной резки

- **Количество врезок (pierces)** — подсчёт точек входа лазера
- **Длина реза** — суммарная длина всех контуров
- **Статистика по слоям** — разбивка по каждому слою
- **Замкнутые/открытые контуры** — автоматическое определение
- **Визуализация точек врезки** — отображение на чертеже

### Раскладка деталей (Nesting)

- **Алгоритм Bottom-Left Fill** — автоматическое размещение деталей на листы
- **Пресеты листов** — стандартные размеры (1000×2000, 1250×2500, 1500×3000, 1500×6000)
- **Произвольный размер листа** — ввод ширины и высоты вручную
- **Зазор между деталями** — настраиваемый отступ (мм)
- **Количество копий** — для каждой детали отдельно
- **Кнопки ±1, ±10** — быстрое изменение количества
- **Кнопка сброса ↺** — возврат количества к 1
- **Автоматический пересчёт** — раскладка обновляется при изменении количества
- **Поворот на 90°** — автоматический для лучшего заполнения

### Визуализация раскладки

- **Сетка листов** — все листы отображаются одновременно (4–5 в ряд)
- **Реальные контуры деталей** — DXF-сущности рисуются внутри bounding box
- **Цветовая кодировка** — каждая деталь своим цветом
- **Имена деталей** — подписи на каждой детали
- **Результаты** — количество листов, % заполнения, общие врезки и длина реза

### Лупа (Zoom Popup)

- **Наведение на лист** — всплывающее окно с увеличенным изображением
- **Zoom колёсиком** — прямо на листе в основном canvas, без перевода мыши
- **Pan мышкой** — перетаскивание в увеличенном окне
- **Zoom к точке курсора** — при скролле на popup
- **Двойной клик** — сброс zoom на исходный
- **Индикатор масштаба** — ×2.3 в углу popup

### Инспектор

- **Клик по сущности** — отображение свойств в боковой панели
- **Тип, слой, цвет** — основные атрибуты
- **Координаты** — геометрические параметры
- **Подсветка при наведении** — hover-эффект

### Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **Ctrl+O** | Открыть файл |
| **F** | Вписать в экран |
| **G** | Сетка вкл/выкл |
| **Esc** | Закрыть раскладку / снять выделение |

### Интерфейс

- **Тёмная тема** — тёмный фон, мягкие цвета
- **Статус-бар** — координаты курсора, zoom, количество сущностей, версия DXF
- **Drag-n-drop** — перетаскивание файлов в окно
- **Адаптивная вёрстка** — подстраивается под размер окна
- **High-DPI** — корректная отрисовка на Retina/4K дисплеях

## Технический стек

- **Vite 5** — сборка и dev-сервер
- **TypeScript 5.3** — строгая типизация
- **Canvas2D** — отрисовка чертежей и раскладки
- **Web Workers** — парсинг DXF в фоне
- **IndexedDB** — хранение настроек
- **Vitest** — 76 тестов (геометрия, bbox, cutting)

## Запуск

```bash
npm install
npm run dev
```

Откроется на http://localhost:5173/

## Сборка

```bash
npm run build
```

Результат в папке `dist/` — готов для деплоя на Vercel, Netlify и т.д.
