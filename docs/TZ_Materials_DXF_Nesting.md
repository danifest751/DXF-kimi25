# ТЗ: База материалов (листовые) + назначение материала для DXF-детали

Версия: 1.0  
Цель: реализовать справочник листовых материалов (актуальных для лазерной резки в РФ) и механизм назначения материала конкретной DXF-детали через модальное окно.

---

# 1. Назначение функции

Пользователь должен иметь возможность:

- выбрать конкретную DXF-деталь (part)
- нажать кнопку «Материал»
- открыть модальное окно
- назначить группе/марке/толщине
- сохранить выбор
- увидеть материал в интерфейсе детали

Материал используется далее для:
- расчёта цены
- расчёта веса
- параметров резки
- nesting

---

# 2. UI/UX

## 2.1 Кнопка

В интерфейсе раскладки или просмотра детали:
- Кнопка: **Материал**
- Расположение: рядом с параметрами выбранной детали
- Доступна только при выбранной детали

---

## 2.2 Модальное окно

### Заголовок
Материал

### Контекст
- Деталь: `<ID / имя>`
- DXF-файл: `<имя файла>` (опционально)

### Поля

1. **Группа материала** (Dropdown)
2. **Марка / тип** (Dropdown, зависит от группы)
3. **Толщина (мм)** (Dropdown, зависит от материала)
4. **Размер листа по умолчанию** (опционально)
5. Checkbox:  
   `Применить ко всем деталям этого DXF`

### Кнопки
- Сохранить
- Отмена

---

# 3. Поведение

- Если у детали уже есть материал — показать текущий выбор
- Если нет — подставить последний использованный пользователем
- После сохранения:
  - материал закрепляется за деталью
  - в UI отображается бейдж:  
    `Сталь Ст3 3мм`
- При выборе "Применить ко всем деталям" — назначить материал всем part без override

---

# 4. Модель данных

## 4.1 Material

```json
{
  "id": "uuid",
  "group": "steel",
  "grade": "Ст3 / S235",
  "thickness_mm": 3,
  "density_kg_m3": 7850
}
```

## 4.2 Назначение детали

```json
{
  "part_id": "uuid",
  "material_id": "uuid",
  "updated_at": "timestamp"
}
```

## 4.3 Дефолты

- user_last_material_id
- dxf_default_material_id (опционально)

# 5. Стартовый справочник материалов (РФ, лазерная резка)

## 5.1 Группы

1. Сталь углеродистая (черная)
2. Нержавеющая сталь
3. Алюминий
4. Оцинкованная сталь
5. Латунь (опционально)
6. Медь (опционально)

---

## 5.2 Материалы и толщины

### 1. Сталь углеродистая
Тип: Ст3 / S235  
Толщины (мм):  
1, 1.5, 2, 3, 4, 5, 6, 8, 10, 12, 14, 16, 20

---

### 2. Нержавеющая сталь
Тип: AISI 304  
Тип: AISI 430  
Толщины (мм):  
0.8, 1, 1.5, 2, 3, 4, 5, 6, 8, 10

---

### 3. Алюминий
Тип: АМг3 / 5052  
Тип: 6061  
Толщины (мм):  
1, 1.5, 2, 3, 4, 5, 6, 8, 10

---

### 4. Оцинкованная сталь
Тип: Оцинкованная (Z)  
Толщины (мм):

---

### 5. Латунь (опционально)
Тип: Л63  
Толщины (мм):  
0.8, 1, 1.5, 2, 3

---

### 6. Медь (опционально)
Тип: М1  
Толщины (мм):  
0.8, 1, 1.5, 2, 3

---

# 6. Размеры листов (Sheet Presets)

- 1000×2000
- 1250×2500
- 1500×3000
- 1500×6000
- 2000×6000

Каждому материалу может быть назначен список рекомендуемых размеров.

---

# 7. Валидации

- Материал обязателен перед запуском расчёта цены
- Толщина обязательна
- Если выбран режим применения ко всем — назначение атомарное (без частичного сохранения)

---

# 8. Acceptance Criteria

1. Пользователь выбирает деталь → нажимает Материал → назначает → сохраняет.
2. Материал отображается в UI детали.
3. Справочник содержит материалы из раздела 5.
4. Возможность расширения справочника через JSON-конфиг.
5. Повторное открытие модального окна показывает текущий материал.
