## 🎨 Концепция дизайна

### Стиль
- **Modern Dark Pro** — тёмная тема для профессиональных CAD-приложений
- **Glassmorphism** — полупрозрачные панели с размытием
- **Neon Accents** — неоновые акценты для активных элементов
- **Minimalist Toolbar** — минималистичная панель инструментов

## 🎭 Анимации и эффекты

### Hover-эффекты
```css
.btn-hover {
  transition: all 0.2s ease;
}

.btn-hover:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 12px rgba(0, 212, 255, 0.2);
}

.btn-hover:active {
  transform: translateY(0);
}
```

### Pulse-анимация для активных элементов
```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.6; }
}

.active-indicator {
  animation: pulse 2s infinite;
}
```

### Fade-in для панелей
```css
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

.panel {
  animation: fadeIn 0.3s ease;
}
```
