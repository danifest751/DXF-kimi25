# Предложения по улучшению UX левой панели (Sidebar)

## Анализ текущего состояния

Текущая левая панель содержит:
- **Шапка**: Заголовок "Файлы и каталоги" + кнопки действий (Выделить все, + Каталог, + DXF)
- **Фильтр каталогов**: Чипсы (chips) для быстрого выбора каталогов
- **Список файлов**: Иерархический список (каталог → файлы) с чекбоксами
- **Подвал**: Статистика резки и переключатель "Врезки"

## Проблемы текущего UX

### 1. Перегруженная шапка
- Три кнопки в одном ряду создают визуальный шум
- "Выделить все" редко используется, но занимает равное пространство
- Нет контекста — неясно, что делает "+ Каталог" без авторизации

### 2. Неявная иерархия каталогов
- Строка каталога кликабельная целиком (фокус), но есть ещё чекбокс (мультивыбор)
- Подсказка в title помогает, но не все пользователи её читают
- Действия "Переименовать/Удалить" видны всегда — визуальный шум

### 3. Нет визуального разделения зон
- Фильтр каталогов (chips) и список файлов сливаются
- Неясно, где заканчивается управление каталогами и начинается работа с файлами

### 4. Отсутствует контекст для гостей
- Кнопка "+ Каталог" не работает без авторизации, но отображается
- Нет объяснения, что каталоги доступны только залогиненным

### 5. Статистика "спрятана" внизу
- Важная информация о врезках и длине реза в подвале
- Нет индикации, что эти данные обновляются при выборе файлов

## Предложения по улучшению

### 1. Реорганизация шапки панели

**Текущая структура:**
```
[Файлы и каталоги]          [Выделить все] [+ Каталог] [+ DXF]
```

**Предлагаемая структура:**
```
[📁 Библиотека]                        [+ 📄 DXF]
────────────────────────────────────────
📂 Все каталоги  •  📂 Каталог А  •  📂 Каталог Б
```

**Изменения:**
- Убрать "Выделить все" из шапки — перенести в контекстное меню или первый элемент списка
- "+ Каталог" показывать только для авторизованных пользователей
- Добавить иконку к заголовку для визуальной идентификации
- Сделать шапку компактнее (высота 44px вместо текущей)

### 2. Улучшение управления каталогами

**Вариант A: Двухуровневая панель (рекомендуется)**

```
┌─────────────────────────────────────┐
│ 📁 Мои файлы           [+ 📄 DXF]  │
├─────────────────────────────────────┤
│ ▼ 📂 Все каталоги (12)    [⚙️]     │
│ ─────────────────────────────────── │
│ • 📂 Проект А              (5 шт)  │
│ • 📂 Проект Б              (3 шт)  │
│ • 📂 Архив                 (4 шт)  │
├─────────────────────────────────────┤
│ 📄 Файлы каталога "Проект А"       │
│ ─────────────────────────────────── │
│ ☐ detail_1.dxf          [🗑️]      │
│ ☐ detail_2.dxf          [🗑️]      │
│ ☑ detail_3.dxf          [🗑️]      │
└─────────────────────────────────────┘
```

**Преимущества:**
- Явное разделение: верхняя часть = навигация по каталогам
- Нижняя часть = файлы выбранного каталога
- Шестерёнка (⚙️) открывает меню управления каталогами (переименовать/удалить)
- Действия с каталогом спрятаны в меню — меньше визуального шума

**Вариант B: Аккордеон каталогов (компактный)**

```
┌─────────────────────────────────────┐
│ 📁 Мои файлы           [+ 📄 DXF]  │
├─────────────────────────────────────┤
│ ▼ 📂 Все каталоги        [+ 📁]    │
├─────────────────────────────────────┤
│ ▼ 📂 Проект А (5)        [⚙️]      │
│   ☐ detail_1.dxf                    │
│   ☐ detail_2.dxf                    │
│ ▶ 📂 Проект Б (3)        [⚙️]      │
│ ▶ 📂 Архив (4)           [⚙️]      │
└─────────────────────────────────────┘
```

### 3. Контекстная адаптация для гостей

**Для неавторизованных пользователей:**

```
┌─────────────────────────────────────┐
│ 📁 Мои файлы (гость)     [+ 📄 DXF]│
├─────────────────────────────────────┤
│ 💡 Войдите через Telegram для       │
│    создания каталогов               │
│    [🔵 Вход]                        │
├─────────────────────────────────────┤
│ ☐ detail_1.dxf                      │
│ ☐ detail_2.dxf                      │
└─────────────────────────────────────┘
```

**Изменения:**
- Убрать кнопку "+ Каталог" для гостей
- Показывать мягкую подсказку о преимуществах авторизации
- Кнопка входа прямо в панели — CTA (call-to-action)

### 4. Улучшенная статистика

**Текущее:** Статистика внизу панели, не всегда видна

**Предлагаемое:** Мини-статистика над списком файлов

```
┌─────────────────────────────────────┐
│ 📁 Мои файлы                        │
├─────────────────────────────────────┤
│ 📊 3 файла • 42 врезки • 12.5м рез  │
├─────────────────────────────────────┤
│ ▼ 📂 Все каталоги                   │
└─────────────────────────────────────┘
```

Или статистику сделать "липкой" (sticky) при прокрутке.

### 5. Улучшенные действия с файлами

**Текущее:** Кнопка удаления появляется при hover

**Предлагаемое:**
- Постоянное меню действий (три точки ⋮) при hover
- Drag-and-drop между каталогами (для авторизованных)
- Контекстное меню по правому клику

```
│ ☐ detail_1.dxf        [⋮] [🗑️]     │
```

### 6. Пустые состояния (Empty States)

**Текущее:** Просто "Нет файлов"

**Предлагаемое:** Полезные пустые состояния

```
┌─────────────────────────────────────┐
│ 📁 Мои файлы                        │
├─────────────────────────────────────┤
│                                     │
│   📭 Пока нет файлов                │
│                                     │
│   Перетащите DXF сюда               │
│   или [выберите файл]               │
│                                     │
│                                     │
└─────────────────────────────────────┘
```

Для каталога без файлов:
```
│ 📂 Проект А (0)                     │
│   ├─ 📭 Перетащите файлы сюда      │
│   └─ или [загрузить в этот каталог]│
```

### 7. Поиск и фильтрация

Добавить строку поиска при >10 файлов:

```
┌─────────────────────────────────────┐
│ 📁 Мои файлы                        │
├─────────────────────────────────────┤
│ 🔍 Поиск в файлах...                │
├─────────────────────────────────────┤
│ ▼ 📂 Все каталоги                   │
└─────────────────────────────────────┘
```
