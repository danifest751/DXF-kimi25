## 🎨 Что было сделано

### 1. Новая цветовая схема "Modern Dark Pro"

**Палитра**:
- **Фон**: Глубокий тёмно-синий (#0a0e1a, #121826, #1a2236)
- **Акценты**: Неоновый голубой (#00d4ff), зелёный (#00ff9d), фиолетовый (#7b2cbf)
- **Текст**: Белый с оттенками серого для иерархии

**CSS переменные**: 50+ переменных для гибкой настройки

### 2. Обновлённые компоненты

#### Toolbar (Верхняя панель)
```css
- Градиентный фон с blur-эффектом
- Увеличенная высота (52px → более просторно)
- Анимированные кнопки с hover-эффектом
- Светящийся логотип с text-shadow
- Shimmer-эффект при наведении
```

#### Cutting Badge (Индикатор резки)
```css
- Неоновая подсветка значений
- Pulse-анимация для активных элементов
- Увеличенные отступы для читаемости
- Тени и свечение
```

#### Pierce Toggle (Переключатель врезок)
```css
- Яркий неоновый зелёный при активации
- Анимация pulse (2s infinite)
- Box-shadow с цветом акцента
- Увеличенный размер точки (10px)
```

#### Sidebar (Боковая панель файлов)
```css
- Градиентный фон с glassmorphism
- Вертикальный акцент слева для активного файла
- Плавное смещение при наведении (translateX)
- Иконки с drop-shadow
- Увеличенные отступы
```

### 3. Анимации и эффекты

#### Переходы
```css
--transition-fast: 0.15s cubic-bezier(0.4, 0, 0.2, 1)
--transition-base: 0.2s cubic-bezier(0.4, 0, 0.2, 1)
--transition-slow: 0.3s cubic-bezier(0.4, 0, 0.2, 1)
```

#### Эффекты
- **Shimmer** на кнопках toolbar (скользящий градиент)
- **Pulse** для активных элементов (переключатель врезок)
- **Glow** для акцентных цветов (text-shadow, box-shadow)
- **Transform** при наведении (translateY, translateX, scale)
- **Backdrop blur** для панелей (glassmorphism)

#### Scrollbar
```css
- Кастомные скроллбары (8px)
- Закруглённые thumb
- Подсветка при наведении
```
